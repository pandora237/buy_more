FAQ

Comment créer un compte client ?

Il suffit de cliquer sur "Inscription" et remplir vos informations personnelles.

J'ai oublié mon mot de passe, que faire ?

Cliquez sur "Mot de passe oublié", puis suivez les instructions pour réinitialiser votre mot de passe.

Quels moyens de paiement acceptez-vous ?

Nous acceptons Mobile Money, Orange Money et Paypal.

Comment suivre ma commande ?

Connectez-vous, allez dans "Mon compte" puis "Mes commandes" pour voir l'état de vos achats.

Comment trouver les promotions ou soldes ?

Allez dans la section "Promotions/Soldes" du site pour voir les offres en cours.

Comment filtrer les produits dans le catalogue ?

Utilisez les filtres disponibles dans le catalogue pour trier par prix, nouveauté ou catégorie.

Comment contacter le service client ?

Allez dans la page "Contact" et remplissez le formulaire pour nous envoyer un message.
